# Course

Course 用来整理学科课程相关的长期学习内容。当前按照三个父级方向归档：Calculus、Physics、Engineering Graphics。

## 课程分类

- [Calculus](./calculus/intro)：极限、导数、积分、梯度和方向导数等基础概念。
- [Physics](./physics/intro)：工程物理课程笔记、实验、公式和应用场景。
- [Engineering Graphics](./engineering-graphics/intro)：制图规范、投影、尺寸标注和图纸整理。

## 维护原则

课程类内容以长期学习和复习为目标，不和项目展示混在一起。缺少真实笔记时先保持简洁，用 TODO 标记待补充内容。
