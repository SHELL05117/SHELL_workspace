# SHELL Workspace

SHELL Workspace 是一个长期维护的个人文档型站点，用来沉淀 robotics、software、AI、systems、music 等方向的资料。

当前 MVP 的边界：

- 静态站点，使用 Docusaurus 构建。
- 代码和 Markdown 文档由 GitHub 管理。
- 第一部署目标是 GitHub Pages。
- 大文件直接放阿里云 OSS，不进入 Git 仓库。
- 中国大陆正式站预留为 OSS + CDN 架构。
